A program egy tetszőleges háromszög adatait számolja ki az oldalak megadása után.
Külön modulban (haromszogoszt) lett definiálva egy Haromszog osztály. 
Az osztály 4 függvényt tartalmaz:
- az init kirajzolja a grafikus felhasználói felületet ehhez tkinter modult használ.
- a h_kerület függvény a háromszög kerületét számolja ki.
- a h_terület függvény a háromszög területét számolja ki.
- a h_derekszog függvény ellenőrzi, hogy a háromszög derékszögű-e.

Az adatok beolvasása során ellenőrzésre kerül, hogy érvényes adatokat adtak-e meg és
a megadott méretű oldalak háromszöget alkotnak-e.
